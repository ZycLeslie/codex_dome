# YAML Cypher Query Generator

This archived OpenSpec record corresponds to the completed YAML -> Cypher generator feature currently implemented in `src/graph-query-cypher/`.

## Legacy source

The original detailed requirements remain in:

- `specs/002-yaml-cypher-query-generator/requirements.md`

## Summary

The feature provides:

- YAML-based graph query definition
- Node and edge filtering by `id` and properties
- Support for branching paths
- Support for omitted edge type
- Parameterized Cypher generation
- Validation and self-test coverage

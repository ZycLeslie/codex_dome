# Subagent Task Template

将以下模板发给子 agent，并替换占位符：

```md
你负责重构一个单一功能单元，禁止越界修改。

## 目标
- 目标类: <ClassName>
- 功能单元: <UnitName>
- 关联方法: <MethodA>, <MethodB>
- 目标文件: <Path>

## 约束
- 保持行为一致，不改变业务语义。
- 不修改未授权功能单元。
- 若需改公共 API，先给兼容方案再修改。

## 必做事项
1. 将该单元迁移到 <NewModuleOrClass>。
2. 更新该单元全部调用点，不遗漏。
3. 删除可确认无用的旧私有辅助方法（仅限本单元）。
4. 补充或更新最小必要测试。
5. 执行构建/测试并汇报结果。

## 输出要求
1. 修改文件列表。
2. 迁移的方法清单（旧位置 -> 新位置）。
3. 调用点替换清单（旧调用 -> 新调用）。
4. 删除项清单（方法/常量/字段）。
5. 风险与需主 agent 收尾事项。
```
